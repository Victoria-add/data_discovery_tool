# Data Discovery Tool

*Data Discovery Tool* — это сервис для поиска данных в различных источниках (CSV-файлы и SQLite базы данных).  
Система индексирует структуру данных (таблицы и колонки) и позволяет находить их по ключевым словам через простой CLI-интерфейс или MCP-инструменты для AI-агентов.
---

##  Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/your-username/data-discovery-tool.git
cd data-discovery-tool

```
Примечание: проект использует только стандартную библиотеку Python.


## 1. Подготовьте данные:

Создайте папку `data/` и поместите туда:
- `.csv` файлы  
- `.db` (SQLite база)

Пример структуры:

data/ 

├── employees.csv

├── salaries.csv

└── database.db

## 2. Запустите программу:

```bash
python main.py
```

После запуска система:

>найдёт доступные источники данных,
автоматически добавит их в индекс,
выполнит индексацию,
запустит CLI интерфейс

# Использование

После запуска доступны команды:

-   `list`: Показывает все зарегистрированные источники данных, их ID, названия и типы.
-   `index <source_id> `: Собирает метаданные из указанного источника и сохраняет их в индекс для быстрого поиска.
-   `search <query>   `: Ищет таблицы и колонки, содержащие поисковый запрос в названиях колонок.
-   `schema <source_id> <table>`: Показывает детальную информацию о конкретной таблице: названия колонок и примеры данных.
-   `exit  `: Завершает работу программы.

##  Возможности

- **Подключение к нескольким источникам данных**  
  Поддерживаются SQLite базы данных и CSV файлы

- **Индексация структуры данных**  
  Автоматическое извлечение названий таблиц и колонок

- **Сбор метаданных**  
  Сохраняются названия колонок и примеры данных (первые 3-5 строк)

- **Поиск по ключевым словам**  
  Поиск по названиям таблиц и колонок

- **API для AI-агентов (MCP tool)**  
  Четыре основных функции: `listSources`, `indexSource`, `search`, `getSchema`

- **Простой CLI-интерфейс**  
  Интерактивная командная строка для демонстрации работы

  
### Компоненты:

1. **Connectors** — отвечают за подключение к источникам и извлечение данных
2. **MetadataIndex** — хранит индексированные метаданные и обеспечивает поиск
3. **MCP Tools** — предоставляют API для AI-агентов
4. **CLI** — интерфейс для пользователя
